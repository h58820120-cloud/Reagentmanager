# 醫療檢驗試劑管理系統
# Medical Laboratory Reagent Management System

Version 1.0.0 | 衛生福利部花蓮醫院

## 📋 系統簡介

一套完整的醫療檢驗試劑出入庫暨批號追溯管理系統，具備以下功能：

- ✅ 完整的使用者管理與角色控制
- ✅ 試劑主檔管理（新增、修改、刪除）
- ✅ 入庫管理（條碼掃描、自動編號、標籤列印）
- ✅ 出庫管理（FEFO先到期先出邏輯）
- ✅ 庫存盤點與差異管理
- ✅ 報廢管理（多種原因分類）
- ✅ 批號追溯系統
- ✅ Code128 & QR Code 條碼生成
- ✅ 自動標籤列印（50mm×30mm）
- ✅ Excel & PDF 報表匯出
- ✅ 操作日誌與審計追蹤
- ✅ 自動備份機制

## 📖 使用指南

### 1. 試劑設定
- 點擊「試劑設定」按鈕
- 點擊「新增試劑」輸入試劑信息
- 必填欄位：代碼、名稱、安全庫存

### 2. 入庫作業
1. 選擇試劑名稱
2. 輸入 LOT 號和有效期限
3. 輸入入庫數量
4. 點擊「入庫」
5. 系統自動生成院內編號和標籤
6. 列印標籤並貼在試劑容器上

### 3. 出庫作業
1. 掃描或輸入院內編號
2. 系統自動帶出試劑信息
3. 輸入出庫數量、使用單位和儀器
4. 點擊「出庫」完成

### 4. 庫存查詢
- 在「查詢」頁籤可以查看當前庫存
- 支援按試劑名稱或 LOT 號查詢
- 顏色標示：綠色(正常) / 黃色(90天內) / 紅色(30天內) / 深紅色(已過期)

### 5. 庫存盤點
- 點擊「盤點」進行實物盤點
- 輸入實際庫存數量
- 系統自動計算差異
- 差異記錄保存在資料庫

### 6. 報廢管理
- 選擇試劑和 LOT 號
- 輸入報廢數量和原因
- 點擊「執行報廢」
- 庫存自動扣除

### 7. 批號追溯
- 輸入院內編號或 LOT 號
- 查看該批試劑的完整生命周期：
  - 入庫記錄
  - 出庫記錄
  - 盤點記錄
  - 報廢記錄

### 8. 報表匯出（管理員功能）
- 支援多種報表類型：
  - 庫存報表
  - 入庫報表
  - 出庫報表
  - 盤點報表
  - 報廢報表
- 支援格式：Excel (.xlsx) / PDF (.pdf)

## 👥 使用者管理（管理員功能）

### 新增使用者
1. 進入「系統設定」
2. 點擊「新增使用者」
3. 輸入使用者信息：
   - 使用者名稱（必填）
   - 密碼（至少6個字符）
   - 真實姓名（必填）
   - 角色（admin / user）
   - 部門（選填）

### 權限說明

**管理員 (admin)**
- 試劑設定：新增、修改、刪除
- 完整的入庫、出庫、盤點、報廢操作
- 使用者管理
- 報表生成
- 系統設定
- 審計日誌查看

**一般使用者 (user)**
- 入庫、出庫操作
- 庫存查詢
- 盤點操作
- 無法刪除資料或修改系統設定

## 🔧 配置說明

編輯 `config.py` 進行系統配置：

```python
# 警示閾值
EXPIRY_WARNING_DAYS = 90  # 90天內黃色警示
EXPIRY_CRITICAL_DAYS = 30  # 30天內紅色警急

# 備份設定
AUTO_BACKUP_TIME = '02:00'  # 每日凌晨2點自動備份
BACKUP_RETENTION_DAYS = 30  # 保留30天內的備份

# 列印機型號
PRINTER_MODELS = ['Zebra ZD420', 'TSC TTP-244 Pro', 'Godex GX430t']
```

系統使用 SQLite3 資料庫，包含以下表格：

| 表名 | 說明 |
|-----|------|
| users | 使用者帳號與權限 |
| reagent_master | 試劑主檔 |
| stock_in | 入庫紀錄 |
| stock_out | 出庫紀錄 |
| inventory | 當前庫存 |
| inventory_check | 盤點紀錄 |
| scrap_records | 報廢紀錄 |
| barcode_records | 條碼紀錄 |
| audit_logs | 操作日誌 |

## 🆘 常見問題

### Q: 如何重置管理員密碼？
A: 刪除 `reagent_system.db` 重新運行程式，系統會重新建立預設帳號。

### Q: 條碼掃描不工作？
A: 
- 確保 USB 條碼槍已連接
- 條碼槍設定為鍵盤模式
- 在入庫或出庫頁面點擊條碼輸入框後掃描

### Q: 如何備份資料？
A: 
- 系統每日凌晨2點自動備份
- 備份文件在 `backups/` 資料夾
- 手動備份：複製 `reagent_system.db` 文件即可

### Q: 如何匯出報表？
A:
- 進入「報表」頁籤（僅管理員）
- 選擇報表類型
- 點擊「匯出 Excel」或「匯出 PDF」
- 選擇保存位置

### Q: 標籤為什麼沒有列印？
A:
- 檢查列印機是否已連接
- 檢查列印機是否已設定為預設印表機
- 確保有列印紙
- 檢查標籤大小是否正確
