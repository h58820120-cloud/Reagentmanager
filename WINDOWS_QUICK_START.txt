🎯 最快啟動方式 (3 步，5 分鐘)

步驟 1️⃣: 下載所有檔案
───────────────────────────────────────────────────────────────
✓ 將所有檔案放到一個資料夾
  例如: C:\reagent-system\

步驟 2️⃣: 選擇執行方式 (二選一)

  【方式 A】雙擊執行 (最簡單，推薦!)
  ───────────────────────────────────────────────
  1. 打開 C:\reagent-system\ 資料夾
  2. 找到 run.bat 或 run_simple.bat
  3. 雙擊執行
  4. 等待應用啟動 (2-3 分鐘首次，之後更快)

  【方式 B】CMD 執行
  ───────────────────────────────────────────────
  1. Windows 鍵 + R
  2. 輸入: cmd
  3. 按 Enter
  
  4. 在命令列中執行:
     cd C:\reagent-system\
     run.bat

  5. 或者，如果出現符號問題，使用:
     run_simple.bat

步驟 3️⃣: 登入系統
───────────────────────────────────────────────────────────────
應用視窗打開後，輸入:
  帳號: admin
  密碼: admin123
  
然後點擊 [登入] 按鈕

❓ 常見問題

【問題】雙擊 run.bat 沒有反應
───────────────────────────────────
原因: Python 未安裝或未在 PATH 中
解決方案:
  1. 安裝 Python 3.12
  2. 確認勾選 "Add Python to PATH"
  3. 重新啟動電腦
  4. 再試一次

【問題】出現 "不應有 &"
───────────────────────────────────
原因: 批處理檔案語法問題
解決方案:
  使用 run_simple.bat 代替 run.bat
  
  右鍵點擊 run_simple.bat
  選擇 "編輯"
  執行它

【問題】CMD 黑視窗立即關閉
───────────────────────────────────
原因: 可能出現錯誤
解決方案:
  1. 右鍵點擊 run.bat
  2. 選擇 "以管理員身份執行"
  
  或在 CMD 中手動執行，便於查看錯誤訊息

【問題】應用無法啟動
───────────────────────────────────
解決方案:
  手動執行命令查看錯誤:
  
  cd C:\reagent-system\
  python main.py
  
  查看錯誤訊息，參考 CMD_EXECUTION_GUIDE.md

📋 系統要求檢查

確認您有以下環境:

Python 版本:
  在 CMD 中執行: python --version
  應該顯示: Python 3.12.x 或更高
  
  如果顯示 "不是內部命令"，需要安裝 Python

磁碟空間:
  至少 500MB 可用空間
  
RAM:
  4GB 或以上 (推薦 8GB)

網路:
  首次安裝時需要網路 (下載依賴)

🔧 完整執行流程 (首次使用)

1️⃣ 準備檔案
   所有檔案都放到一個資料夾

2️⃣ 執行 run.bat
   雙擊 run.bat 或在 CMD 中執行

3️⃣ 等待初始化 (2-3 分鐘)
   系統會自動:
   • 檢查 Python
   • 建立虛擬環境
   • 下載依賴 (PyQt6, SQLAlchemy 等)
   • 初始化資料庫
   • 啟動應用

4️⃣ 看到登入頁面
   輸入帳號: admin
   輸入密碼: admin123

5️⃣ 進入首頁
   看到儀表板和統計資訊

6️⃣ 修改密碼 (推薦)
   進入系統設定修改密碼
